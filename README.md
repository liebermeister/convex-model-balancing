MATLAB code for Convex Model Balancing

Wolfram Liebermeister (2019)

Dependencies:

  o Metabolic Network Toolbox  (https://github.com/liebermeister/mnt)

  o SBMLtoolbox                ( http://sbml.org/Software/SBMLToolbox)
